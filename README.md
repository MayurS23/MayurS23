## 🚀 Featured Projects

### 🔍 [Distributed Search Engine](https://github.com/MayurS23/distributed-search-engine)
> Google-inspired search engine built from scratch in Python

A production-grade system that crawls web pages, builds a **sharded inverted index** across 4 Redis shards, and ranks results using the **BM25 algorithm** (same as Elasticsearch) — all served through a FastAPI REST API.

`Python` `FastAPI` `Redis` `PostgreSQL` `Docker` `aiohttp` `NLTK` `pytest`

- ✅ Async BFS web crawler with `robots.txt` compliance
- ✅ BM25 ranking + TF-IDF scoring (custom implementation)
- ✅ Positional index for exact phrase queries
- ✅ **92 tests** (unit + integration) — all passing

### 🧠 [AI Codebase Intelligence Platform](https://github.com/MayurS23/ai-codebase-intelligence)
> Talk to any GitHub repository like a senior engineer who's read every file

An AI-powered developer tool combining **RAG**, **call graph analysis**, **AST parsing**, and **Claude AI** to let you ask natural language questions about any codebase — and get precise, reference-backed answers.

`Python` `FastAPI` `ChromaDB` `Claude AI` `OpenAI Embeddings` `NetworkX` `Streamlit` `Docker`

- ✅ Ingests any GitHub repo, builds semantic + graph index
- ✅ Traces execution flows from any entry function
- ✅ Multi-language: Python, JS, TS, Java, Go, Rust, C++, and more
- ✅ **50 tests** across 8 modules — all passing

`Python` `FastAPI` `Redis` `Docker` `Lua`
